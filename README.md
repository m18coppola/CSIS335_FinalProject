# CSIS335_FinalProject

## Dependencies
Requires [ffmpeg](https://ffmpeg.org/)


## Directories
`bin/`: binaries live in here  
`lib/`: header-only libs  
`src/`: source code  
`reports/`: project proposal, progress report and paper content

## How to Build 

`make`
  - create all binaries  

`make clean`
  - delete all binaries
